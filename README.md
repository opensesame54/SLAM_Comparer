# SLAM_Comparer
Comparing vSLAM results generated by a ORB-SLAM thingy and a SIFT-SLAM thingy

# SLAM Comparer

## Dependencies

This project requires two external repositories:

1. ORB_SLAM3: Clone from https://github.com/UZ-SLAMLab/ORB_SLAM3
2. Pangolin: Clone from https://github.com/stevenlovegrove/Pangolin

### Setup Instructions

```bash
# Clone required repositories
git clone https://github.com/UZ-SLAMLab/ORB_SLAM3.git
git clone https://github.com/stevenlovegrove/Pangolin.git

# Build ORB_SLAM3
cd ORB_SLAM3
./build.sh

# Build Pangolin
cd ../Pangolin
mkdir build
cd build
cmake ..
make -j4
```
